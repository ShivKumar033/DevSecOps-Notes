## Prompt 1: High-Level Code Understanding

```bash
Act as a senior application security engineer.

Analyze the following source code and explain:
1. What this code does (business logic)
2. Key components and modules
3. Entry points exposed to users
4. Trust boundaries (client vs server vs third-party)

Keep explanations clear and security-focused.
```
✅ Finds: attack surface, logic flow

## Prompt 2: Attack Surface Enumeration

```bash
List ALL user-controlled inputs in this code.

Include:
- HTTP GET/POST parameters
- JSON bodies
- Headers
- Cookies
- JWT claims
- File uploads
- Environment variables

Map each input to where it is used in the code.
```
✅ Finds: hidden inputs, dangerous flows
